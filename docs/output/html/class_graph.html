Detailed Description

A class that represents a graph.

The Graph class is the most important class of the program, since it models the dataset that the developed tasks/algorithms will run on.
Each instance contains all its nodes/vertexes organized in a vector, each one of the nodes containing the respective edges.

NOTE: The terms "node" and "vertex" are used interchangeably.
